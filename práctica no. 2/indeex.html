Control de flujo

  1. ) ¿Cómo funcionan las sentencias condicionales "if", "else" y "switch"?

    2. R= El IF el programa revisa si es correcta y si es correcta hace una accion En el else si la condicion no se cumple entonces hace otra cosa y por ultimo el switch se compara un mismo valor con varias opciones y se ejecuta la que coincida

  2. ) ¿Cuáles son las diferencias entre los bucles "for", "while" y "do...while"?

    3. R= El for se usa cuando sabemos cuantas veces quieres repetir Algo, el while repite una accion mientras mientras la condicion sea verdadera pero puede que no se ejecute nunca y por ultimo do..while hace la misma accion que while repite mientras una condicion sea verdadera, pero se ejecuta al menos una vez, aunque la condicion sea falsa desde el principio

  3. ) ¿Para qué sirven las palabras clave "break" y "continue"?

    4. R= El break detiene el ciclo por completo aunque todavia no haya terminado y por continue salta a la siguiente vuelta del ciclo sin ejecutar el resto de instrucciones

  4. ) ¿Qué es el scope en JavaScript y cómo afecta a las variables?

    5. R= R = El scope en JavaScript es el alcance de una variable, o sea dónde puede usarse. Puede ser global (en todo el programa), de función (solo dentro de una función) o de bloque (con let y const). Esto afecta a las variables porque define si son visibles en todo el código o solo en una parte.
